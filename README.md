# Cuaderno de laboratorio
Este es un cuaderno de laboratorio, ideado para organizar los trabajos realizados en mi estadía en el Laboratorio de Anatomía y Farmacognosia vegetal de la Universidad Nacional Mayor de San Marcos, inciando en 2026.
# Objetivos del cuaderno de laboratorio
- Registrar los procedimientos rutinarios q realice.
- Registrar protocolos nuevos que se ensayen.
- Practicar el uso de R y Quarto para realizar informes.
- Tener backup de la información registrada en forma física.
# Sobre la información registrada
Los datos divulgados en este sitio pueden ser empleados para la práctica de estadística, pues los datos registrados son de casos reales; o en un analisis comparativo realizados por terceros
